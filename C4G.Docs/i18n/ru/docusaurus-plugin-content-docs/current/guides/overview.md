# Обзор руководств

Добро пожаловать в руководства C4G! Эти руководства помогут вам понять и эффективно использовать возможности C4G.

## Доступные руководства

Руководства C4G в настоящее время разрабатываются. Скоро появится подробная документация по:

### Управление конфигурацией

- **Структура конфигурации** - Узнайте, как структурировать Google Sheets для оптимальной генерации конфигураций
- **Парсеры типов** - Понимание и создание пользовательских парсеров типов
- **Генерация кода** - Глубокое погружение в процесс генерации кода

### Интеграция

- **Интеграция с Unity** - Лучшие практики для интеграции C4G в рабочий процесс Unity
- **Настройка CI/CD** - Автоматизация генерации конфигураций в вашем конвейере сборки
- **Контроль версий** - Управление сгенерированным кодом и конфигурациями в Git

### Продвинутые темы

- **Пользовательские парсеры типов** - Создание парсеров для сложных типов данных
- **Модульный дизайн** - Понимание архитектуры C4G
- **Оптимизация производительности** - Советы по оптимизации загрузки и генерации конфигураций

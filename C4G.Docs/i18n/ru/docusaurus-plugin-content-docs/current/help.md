## Часто задаваемые вопросы

### Какие версии Unity поддерживаются?

C4G поддерживает Unity 2019.4 и более поздние версии.

### Могу ли я использовать C4G вне Unity?

Да! C4G доступен как NuGet-пакет .NET Standard 2.0 для автономных приложений.

### Как обращаться с конфиденциальными учетными данными?

Никогда не сохраняйте учетные данные Google OAuth в системе контроля версий. Храните их безопасно и обращайтесь к ним через переменные окружения или безопасное управление конфигурацией.

### Работает ли C4G с файлами Excel?

В настоящее время C4G поддерживает только Google Sheets. Поддержка Excel может быть добавлена в будущих версиях в зависимости от запросов сообщества.

### Как я могу внести свой вклад?

Мы приветствуем вклад! Пожалуйста:

1. Прочитайте [Манифест](https://github.com/vangogih/C4G/blob/master/MANIFEST.md)
2. Проверьте открытые проблемы для хороших первых задач
3. Отправляйте pull request с четкими описаниями

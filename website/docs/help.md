## Frequently Asked Questions

### What Unity versions are supported?

C4G supports Unity 2019.4 and later versions.

### Can I use C4G outside of Unity?

Yes! C4G is available as a .NET Standard 2.0 NuGet package for standalone applications.

### How do I handle sensitive credentials?

Never commit your Google OAuth credentials to version control. Store them securely and reference them via environment variables or secure configuration management.

### Does C4G work with Excel files?

Currently, C4G only supports Google Sheets. Excel support may be added in future versions based on community demand.

### How can I contribute?

We welcome contributions! Please:

1. Read the [Manifest](https://github.com/vangogih/C4G/blob/master/MANIFEST.md)
2. Check open issues for good first issues
3. Submit pull requests with clear descriptions
